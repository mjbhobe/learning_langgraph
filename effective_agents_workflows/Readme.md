### Effective Agent Worflows
This repository is an implementation of workflows illustrated in Anthropic's article [Building Effective Workflows](https://www.anthropic.com/engineering/building-effective-agents).

Each workflow is built using the LangGraph framework.

Entire notes for this project can be accessed [here](https://mirror-feeling-d80.notion.site/Workflow-And-Agents-17e808527b1780d792a0d934ce62bee6)
